# Task 2: Social Media Campaign Performance Analysis

## Objective
Analyze social media campaign data to evaluate engagement, conversions, and spending efficiency.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

## Key Insights
- Instagram campaigns generated the highest engagement and conversions.
- Higher spend does not always guarantee higher conversions.
- LinkedIn showed strong conversion efficiency with moderate spend.
